    3 - connect working device to pc uses usb cable
    + 4 - open project uses Intellij IDEA Community uses *.ipr file, wait all processes/indexing/downloads done, then build project, RESTART IDEA(this create android configuration, before restart you will see nothing), and "run-Run...- android" configuration uses connected testing device.
