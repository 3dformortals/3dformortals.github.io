    All data class properties declared in the primary constructor, have next functionality: 1- equals() / hashCode() pair 1- toString() of the form "User(name=John, age=42)" 1- componentN() functions corresponding to the properties in their order of declaration 1- copy() function 1-destructuring declarations 1- primary constructor default and named arguments
    Requirements of data class : 1-the primary constructor needs to have at least one parameter 1-all primary constructor parameters need to be marked as val or var 1-data classes cannot be abstract , open , sealed , inner 1-data classes may extend other classes 1-to exclude a property from the generated implementations, declare it inside the class body 1-excluded property hidden for toString() , equals() , hashCode() , copy() , componentN() functionality
