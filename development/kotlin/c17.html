• Calling Java code from Kotlin
    • Kotlin code is fully compatible with Java code. weblink weblink

    Calling Java void method from Kotlin file.
    While calling a Java code from Kotlin whose return types is void , it will return Unit in Kotlin file.
    Main.kt file: fun main( args:Array<String> ){ 1 val sum= MyJavaClass.add( 5, 10 ) //printing inside Java class :15 1 println( "printing sum inside Kotlin file: " + sum ) //printing sum inside Kotlin file: kotlin.Unit 0}
    MyJavaClass.java file: public class MyJavaClass { 1 public static void main(String[ ] args){ } 1 public static void add( int a, int b ){ 2 int result = a + b; 2 System.out.println( "printing inside Java class :" + result ); 1 } 0}

    Calling Java int method from Kotlin file.
    While calling a Java code of int type or other (rather than void) from Kotlin file , it returns the result in same types .
    Main.kt file: fun main( args:Array<String> ){ 1 val sum= MyJavaClass.add( 5, 10 ) 1 println( "printing sum inside Kotlin file: " + sum ) //printing sum inside Kotlin file: 15 0}
    MyJavaClass.java file: public class MyJavaClass { 1 public static void main(String[ ] args){ } 1 public static int add( int a, int b ){ 2 int result = a + b; 2 return result; 1 } 0}
