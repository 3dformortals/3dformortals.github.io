    This mind is possible to create libs, for example uses Haxe - The Cross-platform Toolkit , and then compile code to java, that uses inside Kotlin later... on your own risky.
    Android development recommended uses: 1- Android Studio with Kotlin Android Extensions plugin 1- Intellij IDEA Community with Kotlin Android Extensions plugin
    Simple testing is possible uses vscode + Code Runner extension(run kotlin file uses terminal command and show output inside vscode) , and Kotlin Language extension(syntax highlight) , but it is partically support kotlin and at this moment (2018) at least import and object syntaxes not working correct with Code Runner. This mind in vscode using case, at least all code must be placed inside one file, which often named Main.kt in lot of examples.
