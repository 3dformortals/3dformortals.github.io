• pyinstaller create onefile exe
    • PyInstaller official website
    move to target path(with source code) and open console/terminal
    pyinstaller --onefile --noconsole pythoncode.py
    create onefile executable programm pythoncode.exe. Console/terminal will be hidden.
    Supported gui:

    - tkinter (close to crossplatform... little difference with style and size of widgets, and need different events at least for mouse scroll for linux and windows/osx variants... windows/osx event "<MouseWheel>", linux event "<Button-4>" and "<Button-5>" for widget.bind_all("<Eventname>",callbackfunction) syntax)

    - wxPython (not crossplatform code syntax... linux addon gui elements every next to top, windows - every next to back)

    PyQt not supported ... after creating when try to run create errors.
    Django (version 1.8) not supported(2017 tried few times with help/cheats from official pyinstaller site, but no result... the images not reading from sources)
