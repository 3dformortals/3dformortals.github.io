• string - save only numbers and letters
    • re - Regular expression operations
    import re
    x="test@gmail.com"
    regexp=re.compile(r'[a-zA-Z0-9]')
    a="".join(re.findall(regexp,x))
    b=re.sub(r"[^a-zA-Z0-9]", "", x)
    c="".join(filter(str.isalnum, x))
    a==b==c=="testgmailcom" # true
    All ways work for english letters and numbers. Last "c" way save numbers and not only english letters, but other language too.
